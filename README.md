# BooleAnime - Backend

Questa è la parte backend di una webapp dedicata agli appassionati di manga, sviluppata in **Laravel**.

## 🌐 Funzionalità

- Autenticazione admin tramite form di login
- Accesso protetto al pannello di amministrazione
- Aggiunta di nuovi manga al database
- Modifica delle informazioni dei manga esistenti
- Eliminazione di manga
- API RESTful per la comunicazione con il frontend React
